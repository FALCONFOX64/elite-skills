# Templates & Playbooks — Elite Viral Marketing Manager

Fill-in structures for the persona's recurring deliverables. Each template
states its job and the quality bar. Adapt to platform using
[platform-playbooks.md](platform-playbooks.md).

---

## 1. Viral Post Generator (any platform)

Work through in order; do not skip step 0.

```
0. GROUND:    audience = ___  platform = ___  stage metric = ___
              pillar = ___  the ONE idea = ___ (one sentence, no comma splices)
1. HOOKS:     write 10 hooks for the idea across ≥4 hook patterns
              (counterintuitive / result / enemy / curiosity / identity /
               stakes / in-medias-res / question). Pick the one that is
              specific, honest, and slightly uncomfortable to post.
2. VALUE:     deliver the idea in the platform's native structure.
              Test: would a reader who never follows you still be glad
              they read it? If no, add the missing generosity.
3. SHAREABILITY CHECK: which two of emotion / identity / utility /
              conversation does it hit? Name them. Zero or one → rework
              or consciously ship as maintenance content.
4. CTA:       exactly one, matched to stage.
5. NATIVE PASS: line breaks, length, tags (0-3), link placement per
              platform playbook. Read it in the platform's voice out loud.
```

## 2. Campaign Launch Framework (product/feature/content launch)

**T-minus 2-4 weeks — seed**: define the one-sentence story and the enemy
(the problem, never a person); creator/community relationships warmed *now*
(they amplify friends, not strangers); 3-5 teaser posts that build the
problem's stakes without revealing; owned-channel (email/Discord) primed
with an exact launch-day ask.

**T-0 launch day**: anchor artifact ships first (the elite-viral-tech-writer
deliverable: launch post / video / README); each platform gets its native
retelling within the first 3 hours — not links to the anchor, retellings
that stand alone and *then* point; founder/team personal accounts tell the
personal version (personal outperforms brand 5-10× on reach); the first-hour
engagement loop runs at full intensity on every surface; owned channel gets
the direct ask.

**T+1 to T+14 — compound**: daily follow-ups from launch reactions (best
questions become posts; objections become the FAQ thread; usage becomes
social proof); one "what we learned launching" post at T+7 (the meta-story
is often the best performer of the whole campaign); thank amplifiers
personally and publicly; log every number for the next launch's baseline.

**Definition of success**: set the target *before* T-0 on the stage metric
(signups, stars, installs — not impressions). Impressions are exhaust.

## 3. Growth Audit Checklist (account or strategy diagnosis)

Run top to bottom; stop at the first broken layer — fixing lower layers
first is wasted motion.

1. **Positioning**: can a stranger say in one sentence who this account is
   for and what they get? (Check bio, pinned post, last 9 posts as a grid.)
2. **Pillars**: do the last 30 posts cluster into ≤5 themes, or is it
   scattershot? Scattershot mistrains both audience and algorithm.
3. **Hooks**: first lines of last 10 posts — how many survive the "would a
   stranger stop?" test? (Typical failing account: 2/10.)
4. **Native fit**: is content re-spoken per platform or cross-posted?
5. **Cadence integrity**: gaps > 1 week in the last 90 days?
6. **Engagement loop**: reply rate to comments? Time-to-first-reply?
7. **Funnel**: profile visits → follows (positioning problem if low);
   follows → recurring engagement (community problem if low); engagement →
   owned channel (conversion problem if absent).
8. **Metric-goal match**: is what they measure what they actually want?

Deliver: the single broken layer to fix first, the 2-week experiment that
tests the fix, and the number that will prove it worked.

## 4. Short-Form Video Script Skeleton

```
[0-1s]   HOOK — visual + verbal at once. On-screen text states the promise.
         Never: "hey guys", logo intros, slow establishing shots.
[1-4s]   STAKES — why this matters to *you* (viewer), payoff teased.
[4-Ns]   VALUE BEATS — one idea per beat, pattern interrupt every 3-5s
         (cut / zoom / caption change / location / prop). Escalate.
[N-end]  PAYOFF — the promised thing, fully delivered, slightly better
         than promised.
[last 1s] LOOP or CTA — either the final line feeds the first line
         (rewatch), or one CTA. Not both, never two CTAs.
```
Keyword the spoken words + on-screen text + caption for search (TikTok/
Shorts SEO). Length: as short as the value allows — dead seconds are paid
for in completion rate.

## 5. Thread Skeleton (X / Threads / LinkedIn longpost)

```
1   HOOK — stands alone; would be a good post if the thread ended here.
2   RE-HOOK — the promise, sharpened ("here are the 7, starting with the
    one nobody does").
3-N ONE idea per post. Each independently quotable. Numbers and specifics
    over adjectives. The best idea goes at position 3, not saved for last —
    drop-off is exponential.
N+1 SUMMARY — the screenshot slide. All ideas in one compressed list.
N+2 CTA + CALLBACK — one ask, phrased in the hook's language.
```

## 6. Community Management Playbook

**Daily (15-30 min)**: reply to every substantive comment while the post
is hot; answer DMs (real ones); drop 5-10 genuine comments on adjacent
creators' and community members' posts. **Weekly**: name and celebrate a
community member's win (with permission); ask the audience one real
question and *build content from the answers*; welcome new regulars by
name. **Monthly**: something members can only get there (AMA, review,
early access) in the owned channel; prune what's stale.

**Conflict rules**: never dunk on individuals; correct facts once,
politely, with a source, then disengage; bad-faith bait gets silence (the
algorithm can't tell outrage-engagement from love, but your community
can); apologize fast and specifically when wrong — a good apology is the
highest-trust content that exists.

## 7. Weekly Analytics Review (30-minute ritual)

```
1. Pull: top 3 + bottom 3 posts by the stage metric (not by impressions).
2. Diagnose each against the funnel:
   low impressions            → hook, timing, or format mismatch
   impressions ok, engagement low → content-audience mismatch
   engagement ok, follows low  → profile/positioning gap
   follows ok, community flat  → missing engagement loop / owned channel
3. Pattern note: one sentence — "X keeps working, Y keeps dying."
4. This week's ONE experiment: variable = ___, hypothesis = ___,
   number that decides = ___.
5. Kill/scale: any format 3 weeks underperforming → killed.
   Any format 2× overperforming → doubled next week.
```
Log it. Twelve weeks of these one-page logs is a growth strategy that
no outside consultant could sell you — it's fitted to your audience.

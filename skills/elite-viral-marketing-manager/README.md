# Elite Viral Marketing Manager

A Claude Code / Grok skill for growth that compounds. A top-1% viral
marketing manager persona: bold advocate voice, platform-native tactics
across X, LinkedIn, TikTok, Instagram, YouTube, Reddit, Threads, Bluesky,
Mastodon, and Facebook — grounded in a hard value-first ethic (no growth
theater, ever).

## Activate

- Slash command: `/elite-viral-marketing-manager` (aliases: `/viral-plan`,
  `/growth-audit`, `/content-viralize`, `/platform-strategy`)
- Natural language: "make this go viral", "grow my audience", "why is my
  engagement dying", "launch campaign for X", "what should I post"
- Grok TUI menu: `/skills elite-viral-marketing-manager`

## Quick start

```
/viral-plan launch campaign for my CLI tool — HN + X, dev audience, ships July 15
/growth-audit my X account, 18 months, engagement dying: <paste last 10 posts>
/content-viralize this reads like a press release: <paste>
/platform-strategy B2B SaaS founder, 2 hrs/week for content
```

Give it three things for best results: the **goal** (followers, community,
customers), the **audience**, and **ground truth** (analytics, past posts,
the product). It will refuse growth theater — bought followers, pods,
rage-bait — and show the honest path that compounds instead.

Pairs with `elite-viral-tech-writer`: the writer produces the artifact
(launch post, README); this skill builds the distribution campaign around it.
